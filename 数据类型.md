>而 b、c、d 输出结果不变
## 二、sizeof关键字

sizeof 可用于计算数据类型所占内存大小

格式：sizeof（数据类型 / 变量）
```
#include <iostream>
using namespace std;
int main()
{
    short a = 10;
    int b = 20;
    long c = 30;
    long long d = 40;

    cout << "short占用内存空间为:" << sizeof(short) << "字节" << endl;
    cout << "int占用内存空间为:" << sizeof(int) << "字节" << endl;
    cout << "long占用内存空间为:" << sizeof(long) << "字节" << endl;
    cout << "long long占用内存空间为:" << sizeof(long long) << "字节" << endl;
    cout << "short占用内存空间为:" << sizeof(a) << "字节" << endl;
    cout << "int占用内存空间为:" << sizeof(b) << "字节" << endl;
    cout << "long占用内存空间为:" << sizeof(c) << "字节" << endl;
    cout << "long long占用内存空间为:" << sizeof(d) << "字节" << endl;

    return 0;
}
```
## 三、浮点型

默认情况下输出一个小数，会显示出6位有效数字

单精度  float     将小数赋值给变量时小数末尾要带 f

双精度  double
```
#include <iostream>
using namespace std;
int main()
{
   float a = 3.15f;
   cout << a << endl;
   double b = 3.16;
   cout << b << endl;
   float c = 3.1415926f;
   cout << c << endl;
   double d = 3.1415926;
   cout << d << endl;
   cout << "float类型的变量占用字节数：" << sizeof(a) << endl;
   cout << "double类型的变量占用字节数：" << sizeof(b) << endl;

   return 0;
}
```
## 补充：科学计数法
```
#include <iostream>

int main()

    return 0;
}
```
## 四、字符型

字符型变量用于显示单个字符

格式：char ch = ' a '

注意：单引号内只能有一个字符，不可以是字符串
```
#include <iostream>
using namespace std;
int main()
{
   char ch = 'A'; // 字符'A'的ASCII码是65
   cout << ch << endl;
   cout << int(ch) << endl; // 强制类型转换，将字符转换为对应的整数ASCII码
   cout << "char字符型变量占用的字节数：" << sizeof(ch) << endl;
   return 0;
}
```

## 五、转义字符

转义字符用于表示一些不能显示出来的ASCII字符

<img width="773" height="545" alt="0321cd7f-b8b7-49c8-bee1-b26d3a05e9a4" src="https://github.com/user-attachments/assets/88b0886d-3abe-4cce-8ad0-a3412ea3f4b0" />

常用的有：\n   \\   \t
```
#include <iostream>
using namespace std;
int main()
{
   cout << "hello world\n";
   cout << "hello today\n";
   cout << "\\" << endl;
   cout << "\\\\" << endl;
   cout << "aaaa\thelloworld" << endl;
   cout << "aa\thelloworld" << endl;
   cout << "aaaaaaa\thelloworld" << endl;
   cout << "aaaaaaaa\thelloworld" << endl; 
   
   // \t ：长度8以内对齐，大于等于8以后直接跳到下一个制表符位

   return 0;
}
```

## 六、字符串型

字符串型用于表示一串字符

两种风格：

### 1.C风格字符串：char  变量名[ ] = “ 字符串值 ”
```
#include <iostream>
using namespace std;
int main()
{
   char str1[] = "Hello";
   cout << str1 << endl;
   return 0;
}
```
### 2.C++风格字符串：string  变量名 = “ 字符串值 ”
```
#include <iostream>
using namespace std;
int main()
{
   string str2 = "world";
   cout << str2 << endl;
   return 0;
}
```
## 七、布尔类型

布尔类型占一个字节大小，表示真或假的值，因此只有两个值

true   真（本质是1）

false  假（本质是0）
```
#include <iostream>
using namespace std;
int main()
{
   bool flag = true;
   cout << flag << endl;                             // 输出1，表示true
   flag = false;
   cout << flag << endl;                             // 输出0，表示false
   cout << "size of bool: " << sizeof(bool) << endl; // 输出bool类型的大小，通常为1字节

   return 0;
}
```

## 八、数据输入

作用：用于从键盘获取数据

关键字：cin

格式：cin >> 变量​​​​
```
#include <iostream>
using namespace std;
int main()
{
   int a = 0;
   cout << "请输入一个整数:" << endl;
   cin >> a;
   cout << "该整数加10后的值为:" << a + 10 << endl;

   double b = 0.0;
   cout << "请输入一个小数:" << endl;
   cin >> b;
   cout << "该小数加10后的值为:" << b + 10 << endl;

   char ch = '0';
   cout << "请输入一个字符:" << endl;
   cin >> ch;
   cout << ch << endl;

   string str = "0";
   cout << "请输入一个字符串:" << endl;
   cin >> str;
   cout << str << endl;

   bool flag = false;
   cout << "请输入一个布尔值(0或1):" << endl;
   cin >> flag;
   cout << "该布尔值取反后为:" << !flag << endl;

   return 0;
}
```
